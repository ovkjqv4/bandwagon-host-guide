## 三、搬瓦工限时秒杀的补货规律与抢购技巧

### 1. 补货规律：不是你想买，想买就能买

从近几年公开的补货记录来看，搬瓦工的补货节奏大致是：

- **频率**：一般 1 周到半个月补货一次，偶有大促节点（双十一、黑五）会密集补货。
- **持续时间**：每次补货通常撑 1—2 天，热门套餐（比如 THE PLAN、CN2 GIA-E 限量版）经常在放出来几小时内就被秒空。
- **放货时段**：北京时间上午或中午相对常见，但偶尔也会在凌晨放货，没有特别固定的规律。
- **特殊节点**：搬瓦工近几年的传统是 **双十一**（11 月上旬）比黑五更慷慨，会放出全场 11% 循环折扣码，同时配合限量版补货，是全年最容易捡便宜的时间窗口。

### 2. 抢限量版，关键不是手速，是"信息差"

我以前一直以为是手快有手慢无，后来才发现真正决定能不能抢到的，是**你能不能在补货放出来的几分钟内就知道这件事**。

主流的补货通知渠道有这么几类，建议至少叠加两个：

1. **Telegram 频道**：响应速度最快，通常补货几分钟内就会有推送。搬瓦工官方有一个英文 Telegram 频道，第三方中文网也有同步频道。
2. **QQ 补货通知群**：搬瓦工中文网维护了多个全员禁言的补货通知 QQ 群（比如 697178487、554576821、451796455 等），只在补货时发消息，不会被打扰。
3. **库存监控页面**：搬瓦工官方有一个 stock.bwg.net 的库存监控页面，每 5 分钟左右刷新一次所有套餐的库存状态，可以作为兜底方案。
4. **邮件订阅**：通过 Google Groups 订阅 bandwagonhost 邮件列表，也能收到补货通知，但延迟会比 Telegram 和 QQ 大。

### 3. 秒杀成功的关键操作

收消息之后真正开抢时，几个小细节会影响成败：

- **提前注册账号并完成邮箱验证**：限量版套餐几秒钟一个库存，临时注册就来不及。
